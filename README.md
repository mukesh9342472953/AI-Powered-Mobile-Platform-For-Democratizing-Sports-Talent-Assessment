# AI Powered Mobile Platform For Democratizing Sports Talent Assessment

An AI-driven platform designed to identify, assess, and rank sports talent using video-based performance analysis.

## Features
- User Registration & Login
- Athlete Dashboard
- Coach Dashboard
- Recruiter Portal
- Video Upload & Assessment
- Leaderboards
- Events Management
- Profile Management
- SQLite Database
- Node.js Backend
- Responsive Mobile UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite
